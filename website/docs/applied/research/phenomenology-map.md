# The Phenomenology Map: experience → observable → test

:::info What this document is
UHM makes consciousness **measurable**: every state of $\Gamma$ has a small set of cheap observables — purity $P$, reflection $R$, integration $\Phi$, the consciousness measure $C=\Phi\times R$, $E$-coherence $\mathrm{Coh}_E$, the [Gap](/docs/core/dynamics/gap-operator), the cognitive temperature $T_{\mathrm{eff}}$, and the differentiation $D_{\mathrm{diff}}=e^{S}$. This document does one thing, honestly: it maps **lived experience** onto those observables, and for each mapping states **how to test it** and **what its status is**. It also draws the strict line for *anomalous* reports — the theory reads the **experience** (a fact about the self-model), and it does **not** assert the reality of any external entity behind it. Where UHM has no mechanism, the entry is marked **[untestable / outside the theory]**, not decorated.
:::

## 1. The mapping principle {#принцип}

Nothing here is a new postulate. UHM already fixes the observables (they are functions of $\Gamma$) and the thresholds ($P>2/7$, $R\geq1/3$, $\Phi\geq1$, $D\geq2$). A *lived state* enters the formalism through the [Enc functor](/docs/applied/coherence-cybernetics/sensorimotor#теорема-кодирование-среды) — perception writes the world into $\Gamma$ — so a report of experience is, in the theory, a report about a **region of the observable space**. The map below is therefore not a metaphor: it is the claim that a given experience corresponds to a specific signature, and that measuring the signature (via the calibrations of the [Measurement Methodology](/docs/applied/coherence-cybernetics/measurement)) tests the claim.

The single most important discipline: a signature is a statement about the **holon's own state**, never about a world beyond it. This is what keeps the map falsifiable rather than a license to explain anything.

## 2. The map {#карта}

| Experience | UHM signature | How to test | Status |
|---|---|---|---|
| Wakeful consciousness | $P>2/7$, $\Phi\geq1$, $C=\Phi R>0$ | PCI $\approx0.31$ threshold | **[falsifiable]** |
| Deep sleep / anaesthesia | $P<2/7$ (loss of viability), $\mathrm{Gap}\to1$ | PCI collapses | **[falsifiable]** |
| Meditation / flow | low $T_{\mathrm{eff}}$ (cognitive temperature) | EEG spectral slope, HRV | **[falsifiable]** |

| "Grokking" / sudden understanding | $\Phi$ crosses $1$ (L1→L2 ignition) | a measurable, dated transition | **[falsifiable]** |
| Psychedelic / ego-dissolution | $\uparrow D_{\mathrm{diff}}=e^{S}$, $P$ toward the window edge | brain-signal entropy $\uparrow$ (see §3) | **[falsifiable]** |

| Déjà vu | self-model prediction leads input | [temporal consciousness](/docs/consciousness/phenomenology/temporal-consciousness) | [interpretation] |

Every row in the first block is a *number against data*, not a mood. The value of the map is precisely that it can be wrong.

## 3. The entropic-brain bridge {#энтропийный-мозг}

The strongest and most independent test comes from work UHM did not author. Carhart-Harris's **entropic brain hypothesis** (2014) proposes that psychedelic and other primary states raise the **entropy** of spontaneous brain activity (measured, e.g., as Lempel–Ziv complexity of EEG/MEG), and that ordinary waking cognition sits at a *self-organised critical* boundary between order and disorder.

This maps directly onto UHM. The differentiation measure is $D_{\mathrm{diff}}=\exp(S_{\mathrm{vN}})$ — the exponential of the von Neumann entropy of $\Gamma$. So "increased brain entropy" is, in the formalism, **increased $D_{\mathrm{diff}}$** — the state moving *up* toward the top of the viability window, and, if pushed far enough, *out* of it (past $P=3/7$ toward $I/7$, where $S\to\ln7$ and viability is lost). The entropic brain's "edge of chaos" is, exactly, UHM's **viability boundary** $P=2/7$; its claim that consciousness lives near criticality is UHM's claim that the conscious window $(2/7,3/7]$ sits between the frozen ($P\to1$) and the dissolved ($P\to1/7$) extremes.

This is more than an analogy, and it connects to a theorem: the [entropy law T-271](/docs/physics/cosmology-phys/origin#t-271) makes the dissipator the entropy source and regeneration the sink, so a psychedelic that *weakens regeneration* (or strengthens the dissipative channel) is predicted to raise $S$ — precisely the entropic-brain observation. **Status: [С] (a measurement bridge, needing the same $S$↔brain-signal calibration as PCI↔$P$) and [falsifiable].** It is a rare case where an existing, replicated empirical program is the natural reading of a UHM invariant.

## 4. Anomalous phenomena — the honest framework {#аномалии}

Reports of altered, "contact," and boundary experiences are data about **experience**. UHM can read them; it must not inflate them. The rule is one line: **map the report to what the theory predicts about the self-model, and never confuse a reading of the experience with confirmation of an entity.**

| Reported phenomenon | UHM reading (about the experience) | Status |
|---|---|---|
| Ego-dissolution / unity | $P$ near the window edge, high $\mathrm{Coh}_E$, $\uparrow D_{\mathrm{diff}}$ | **[falsifiable]** (entropic brain, §3) |
| Sense of presence / "contact" | an aberration of the **self-model** — a fall in $R_\varphi$ (self-model quality) and/or a Gap anomaly. UHM reads it as a phenomenon *of the self-model*, **not** evidence of an entity | **[interpretation]** — does *not* confirm an entity |
| Déjà vu | the self-model's prediction momentarily precedes the input | **[interpretation]** |
| Out-of-body / autoscopy | a mismatch in the sensorimotor [Enc](/docs/applied/coherence-cybernetics/sensorimotor) of the body schema | **[interpretation]** |
| "Rifts" / fields from other realms | **no UHM mechanism** — the theory predicts no location-bound "quantum otherworldly" fields | **[untestable / outside the theory]** |

The last row is the integrity test. UHM is a formal theory with strong falsifiability; it earns that by *declining* to explain what it has no mechanism for. Reading an experience as a self-model event is legitimate and often falsifiable; asserting an external entity or field from the same report is neither, and the map says so plainly.

## 5. The line {#линия}

- **What the theory does:** predict the observable signature of a class of experience, and hand the experimenter a calibration and a falsification criterion.
- **What it does not do:** certify that any felt presence, entity, or "field" is real. The felt presence is real *as an experience* (a self-model state); its external referent is not a UHM claim.

This is not caution for its own sake. A map that can explain any report explains nothing; the value of this one is that most of its rows can be **measured**, one row (§3) already meets an independent empirical program, and the rows it cannot test it labels as such.
