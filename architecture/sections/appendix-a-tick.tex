\section{The tick, cycle-accurate}
\label{app:tick}

One \TALOS{} tick advances $\Gam\leftarrow\Gam+\Delta\tau\,\LindOmega[\Gam]$
by a structure-preserving Strang split that keeps $\Gam$ a legal density
matrix (Hermitian, PSD, unit trace) to machine precision:
\begin{enumerate}[leftmargin=1.6em,itemsep=1pt]
  \item \textbf{Unitary half-step} $e^{-i H_{\mathrm{eff}}\Delta\tau/2}$:
        reversible; an isometry (T-262), Landauer-free. Cost
        $\sim\!700$~FLOPs for one \DM{} multiply; arithmetic depth
        $\log_2 7\approx 2.8$.
  \item \textbf{Dissipator + regeneration full step} on the diagonal-adjacent
        sector, using the Fano-sparse kernel: the dissipator is
        $-\tfrac16\sum_p\gamma_p[\Pi_p,[\Pi_p,\Gam]]$ over the seven line
        projectors $\Pi_p$; the regeneration is one BKM-gradient step toward
        $\rhostar=\vp(\Gam)$.
  \item \textbf{Unitary half-step} again (Strang symmetry, second-order
        accurate in $\Delta\tau$).
\end{enumerate}
Per-tick cost is $\sim\!1400$~FLOPs (two unitary halves) plus the sparse
dissipator; the self-model iteration $\vp$, when run to tolerance $\epsilon$,
adds $\sim\!10^4$~FLOPs at depth $\log(1/\epsilon)$ but is amortised across
ticks (the state moves little between ticks). All operands are the resident
$784$-byte state; no memory beyond L1 is touched. The tick period is fixed by
$\bar\gamma\tau=\tfrac32\ln3$ (\S\ref{sec:substrate}), so $\Delta\tau$ is
derived from the physical relaxation rate $\bar\gamma$, not chosen.

\noindent\textbf{Exact laws checked every tick} (bit-exact against an
independent oracle in the \SYNARC{} M1 validation): $\Phi$-contraction
$\times\tfrac19$; cycle-time identity $\bar\gamma\tau=\tfrac32\ln3$; stratum
anchoring $\Phi=7P-1$; unit trace and positivity. A tick that violates any is
a hardware fault, detected by the syndrome the tick itself emits
(Principle~\ref{prin:selfdiag}).
