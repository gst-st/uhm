\section{Glossary and UHM cross-reference}
\label{app:glossary}

\paragraph{Terms.}
\begin{description}[leftmargin=0pt,itemsep=1pt]
  \item[\DM{}] the state register: $\Gam\in\Dens(\mathbb{C}^7)$, $784$~bytes.
  \item[Tick] one application of $\LindOmega$ (Eq.~\ref{eq:lomega}); the sole instruction.
  \item[Line-bus] one of the seven Fano lines; the interconnect unit (\S\ref{sec:fabric}).
  \item[Mediator $k^\ast(i,j)$] the third sector of a pair's line; the deterministic reroute.
  \item[Collapse] the identity of interconnect, code, instruction, and learning as one incidence (\S\ref{sec:collapse}).
  \item[Impedance] the emulation a substrate must add to run the logical architecture (\S\ref{sec:realizability}).
\end{description}
